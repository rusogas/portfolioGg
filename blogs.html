app flask-tableros

🚀 aplicación creada con Python, Flask, HTML y CSS 🚀 La idea principal de la app es tener varios tableros interactivos que permitan visualizar y analizar datos desde una base de datos. Por ejemplo, en uno de los tableros se realiza un análisis de ventas y se muestran los resultados a través de gráficos generados con Matplotlib. 📊 Gracias a esta aplicación, es posible: Conectar diferentes bases de datos. Realizar análisis de datos como ventas, productos más vendidos, y más. Visualizar los resultados de manera clara y efectiva mediante gráficos de barras, tortas y otros. Este proyecto me permitió combinar el poder de Flask para la creación de aplicaciones web y las capacidades de Matplotlib para el análisis visual de los datos. 💻📈
